<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| npm registry → local node_modules | Transitive package resolution; override pins constrain which package version is installed |
| package-lock.json → CI build | The lock file is committed and controls what CI installs; a tampered or regenerated lock is the supply-chain attack surface |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-01 | Tampering | npm overrides resolution | mitigate | Pin to minimum safe version ranges (^18.1.1, ^0.2.6, ^7.0.5) rather than exact pins to allow patch-level auto-updates; verify via npm ls after install |
| T-16-02 | Tampering | serialize-javascript@7 (major bump in mocha tree) | mitigate | Mandatory mocha smoke-check post-install; rollback procedure defined in Task 2 if runner breaks |
| T-16-03 | Elevation of Privilege | @hapi/wreck <18.1.1 (SSRF, redirect-following vuln) | mitigate | Override to ^18.1.1 closes the Moderate runtime alert (#197); this is the only runtime-scope vulnerability in this batch |
| T-16-04 | Denial of Service | tmp <0.2.6 (insecure temp file creation) | mitigate | Override to ^0.2.6 closes the High dev-scope alert (#198) |
| T-16-05 | Remote Code Execution | serialize-javascript <7.0.3 (prototype pollution) | mitigate | Override to ^7.0.5 closes the High + Moderate dev-scope alerts (#147, #195); mocha smoke-check required |
| T-16-06 | Elevation of Privilege | uuid #194 (transitive uuid@8 in nyc/jest-junit) | accept | Deliberately deferred — dev-only moderate, 3-major version jump risks breaking nyc/jest-junit; regression risk outweighs severity. Documented as deferred-dev-only in REQUIREMENTS.md |
| T-16-SC | Tampering | npm install installs new packages | mitigate | All three packages are existing transitives being version-bumped only (no new packages introduced); no Package Legitimacy Audit gate required — overrides target packages already in the tree |
